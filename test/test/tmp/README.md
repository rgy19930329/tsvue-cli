



# React-Koa2-SSR开发框架
### 更新日志
> 无

### 特点简介
+ React客户端JS框架
+ Koa2服务器端JS框架
+ 服务器端渲染
+ React Koa2 客户端服务器端同构
+ 优化首屏打开速度
+ SPA
+ Redux数据流状态管理
+ Wepack打包
+ 支持ES6/+/7

> 不面向C端也可不适用同构框架，查看可移步[这里](https://github.com/walkstreet/react-koa2-webpack)

### 框架依赖简介
+ React v16.0.0
+ Koa v2.4.1
+ React-router-dom v4.2.2
+ Redux v3.7.2
+ Axios v0.17.1
+ Webpack v3.8.1

### 构建命令
```
# 安装依赖
npm install

# 开发环境启动
npm run start

# 打生产包
npm run build

# 服务器端部署
npm run prd
```
